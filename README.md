В ходе проекта проведено геометрическое моделирование основных деталей узла в среде Siemens NX: ступицы переднего колеса, поворотного кулака и привода переднего колеса с наружным шарниром равных угловых скоростей. На базе разработанных моделей выполнена сборка узла с заданием всех сопряжений, обеспечивающих корректное взаимное положение деталей. Для каждой детали учтены посадочные поверхности под подшипники, шлицевые соединения, резьбовые участки и технологические элементы.

Расчётная часть работы включает определение нагрузок, действующих на колёсно-ступичный узел в наиболее тяжёлом эксплуатационном режиме — экстренном торможении с одновременным наездом на неровность дорожного покрытия. Вертикальная нагрузка на колесо определена с учётом перераспределения веса автомобиля при торможении и динамического коэффициента, возникающего при ударе о препятствие. Тормозной момент рассчитан исходя из максимального сцепления шины с дорогой.

На основании полученных нагрузок выполнен расчёт ступичных конических роликовых подшипников 32008Х на динамическую и статическую грузоподъёмность по ГОСТ 18855-2013. Определены радиальные и осевые составляющие нагрузки на каждый подшипник с учётом схемы установки «в распор». Расчётный ресурс внутреннего подшипника составил более 300 тысяч километров пробега, что значительно превышает нормативные требования.

Проведён прочностной расчёт ступицы методом конечных элементов в программном комплексе ANSYS Workbench. Расчётная модель включала импортированную геометрию, конечно-элементную сетку на основе тетраэдральных элементов второго порядка и граничные условия, соответствующие экстренному торможению. Анализ распределения эквивалентных напряжений по Мизесу показал, что максимальные напряжения не превышают 67 МПа, а запас прочности по пределу текучести составляет 5,7, что свидетельствует о значительном резерве несущей способности детали. Максимальные упругие перемещения не превышают 0,03 мм.

Дополнительно выполнен аналитический расчёт эвольвентного шлицевого соединения ступицы с валом привода по ГОСТ 6033-80. Проверка соединения на смятие и срез подтвердила его работоспособность с запасом прочности 1,7 и 1,6 соответственно.

По результатам проектирования разработана конструкторская документация в соответствии с требованиями ЕСКД: чертежи общего вида, сборочный чертёж колёсно-ступичного узла, рабочие чертежи сопряжённых деталей и спецификация.
